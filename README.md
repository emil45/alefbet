# Lepdy - Hebrew Learning for Kids

A fun, interactive web application that helps children learn Hebrew letters, numbers, colors, shapes, animals, and food through colorful cards and educational games.

**Live site: [lepdy.com](https://lepdy.com)**

## Features

- **Interactive Learning Cards** - Tap to hear Hebrew pronunciation with engaging visuals
- **Multiple Categories** - Letters (אותיות), Numbers (מספרים), Colors (צבעים), Shapes (צורות), Animals (חיות), Food (אוכל)
- **Educational Games** - Memory match, guessing games, Simon says, and more
- **Multilingual** - Hebrew, English, and Russian interfaces
- **Mobile-Friendly** - Works on phones, tablets, and desktop
- **Kid-Safe** - No ads, no in-app purchases, no external links

## Tech Stack

- [Next.js](https://nextjs.org/) 16 with App Router
- [React](https://react.dev/) 19
- [Material UI](https://mui.com/) 7
- [next-intl](https://next-intl-docs.vercel.app/) for internationalization
- [Firebase](https://firebase.google.com/) for leaderboards
- [Playwright](https://playwright.dev/) for E2E testing

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
